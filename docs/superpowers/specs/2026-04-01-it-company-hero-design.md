# Aether Intelligence — IT企業トップページ Hero デザインスペック

## 概要

架空のエンタープライズAI企業「Aether Intelligence」のトップページ。フルスクリーンのHeroセクション1ページで構成。メッシュグラデーションアニメーションとリッチなテキスト演出で「感動するレベル」の体験を提供する。

## ルート

`/it-company` — `src/app/it-company/page.tsx`

## デザインスタイル

- **テーマ:** ライト & エンタープライズ
- **背景:** グレー〜ホワイトのグラデーション（`#f8fafc` → `#f1f5f9`）
- **アクセント:** パープル（`#7c3aed`）× ブルー（`#2563eb`）のグラデーション
- **タイポグラフィ:** Geist Sans（プロジェクト既存）、ウェイト300〜700
- **印象:** 落ち着き、高級感、信頼性

## ページ構成

### ナビゲーションバー
- 左: ロゴ（Aether Intelligence）
- 右: ナビリンク（Products / Solutions / Research / Company）+ CTAボタン
- 背景: 透過 → スクロールでblur付きソリッド（ただしHeroのみなのでスクロールは最小限）

### Heroセクション（フルスクリーン 100vh）
- **キャッチコピー:** 大きなタイポグラフィ（2〜3行）、AI/知性に関するメッセージ
- **サブテキスト:** 1〜2行の補足説明
- **CTAボタン:** 2つ（プライマリ: グラデーション塗り / セカンダリ: ゴースト）
- **テキスト配置:** 中央揃え

## アニメーション仕様

### メッシュグラデーション（背景）
- 3〜4個のカラーブロブ（パープル、ブルー、シアン系）
- `radial-gradient` + `filter: blur()` で実現
- CSSアニメーション（`@keyframes`）でゆっくり位置・サイズが変化
- 周期: 15〜25秒、各ブロブで異なるタイミング
- グリッドオーバーレイなし

### テキスト演出
- キャッチコピー: 文字単位のスタガーフェードイン（ブラー→クリア）
- サブテキスト: フェードイン + 上方向スライド（遅延あり）
- CTAボタン: フェードイン + スケール（さらに遅延）
- anime.jsを使用して精密なタイミング制御

### ナビゲーション
- 初期: 透明 → フェードインで表示

## 技術スタック

- Next.js 16 / React 19 / Tailwind CSS v4
- anime.js（テキストアニメーション）
- CSSカスタムプロパティ + `@keyframes`（メッシュグラデーション）
- `"use client"` ディレクティブ（アニメーション用）

## ファイル構成

```
src/app/it-company/
  page.tsx    — メインページコンポーネント（単一ファイル）
```

Heroセクションのみのシンプルな構成のため、コンポーネント分割は不要。`page.tsx`に全てを含める。

## 成功基準

- フルスクリーンHeroがビューポート全体を占める
- メッシュグラデーションが滑らかに動き続ける
- テキストのブラー→クリア演出が段階的に再生される
- モバイルでもパフォーマンスが良好
- 既存のホームページ（`/`）のarticlesリストに追加可能な構造
