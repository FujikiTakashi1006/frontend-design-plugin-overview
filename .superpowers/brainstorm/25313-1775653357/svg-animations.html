セクション別 SVGアニメーション

7 SECTIONS — CONCEPT VISUALIZATION

01 導入：問いを投げる

ソーシャルネットワーク図

ヒーロー部分で承認済み。中心ノード（自分）から周囲の人々へ接続線が伸び、パルスが波紋のように広がる。

接続線の描画 → ノード出現 → パルスリング（ループ）
02 脳が巨大化した理由

社会脳 × ダンバー数

脳のシルエットの内側と外側に、同心円状の群れリングが広がる。人を表すドットが順次出現し、最後に「150」のダンバー数が表示。

脳シルエット描画 → 群れリング拡張（3段階）→ 人ドット出現 → 「150」フェードイン
03 性格の正体：戦略の切り替え

ペルソナ切り替え（if-thenマスク）

3つの表情の異なるマスクが順次出現。上司の前では真面目、友人の前では笑顔、初対面では慎重。切り替え矢印で「if→then」パターンを視覚化。

マスク1出現 → 矢印 → マスク2出現 → 矢印 → マスク3出現 → 「if→then」ラベル
04 進化が組み込んだ同調装置

同調圧力の波と逸脱エラー

集団の基準線を表す波に沿ってドットが整列。1つだけ外れたドットに「ERROR」シグナルが点滅。脳が逸脱をエラーとして処理する仕組みを視覚化。

波線描画 → ドット整列 → 逸脱ドット出現＋ERRORパルス（ループ）
05 伝染する感情・行動・人生

社会的伝染の波紋

中心ノードから波紋が広がり、友人→友人の友人→3次接続へと影響が伝播。各層のノードが順次アクティブに。フラミンガム研究の「57%」を表示。

波紋リング（ループ）→ 第1層アクティブ → 第2層アクティブ → 「57%」フェードイン
06 自尊心の正体：ソシオメーター

ソシオメーター（受容度ゲージ）

半円のゲージメーター。針が左（低い自尊心・否定的環境）から右（高い自尊心・肯定的環境）へ振れる。環境のアイコンが左右に表示。

ゲージ背景 → 針が左から振れ始め → 否定的環境アイコン → 針が右へ → 肯定的環境アイコン
07 結論：自分自身を再構成する

ネットワーク再構成

左側の古いネットワーク（薄い線）から、右側の新しいネットワーク（鮮明な線）へ。中心の自分は変わらず、周囲の接続先が入れ替わる。「自分を変える＝周りを変える」の視覚化。

古いネットワーク表示（薄い）→ 新しい接続線が描画 → 新しいノード出現 → 「re:connect」ラベル
